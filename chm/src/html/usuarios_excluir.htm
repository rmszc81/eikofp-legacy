Excluir meu usuário

Onde encontrar:
Menu Principal -> Usuários -> Excluir meu usuário.

Descrição:
Permite que o usuário atual logado exclua todos os seus dados permanentemente.

Imagem da tela:

Itens da tela :
Senha: Campo obrigatório, mínimo de 04 caracteres.

Comandos da tela :
Excluir: Executa a validação da senha preenchida e em seguida elimina todos os dados do usuário atual logado.
Cancelar: Cancela a operação, retornando para a tela anterior.

Observações:
Ao clicar no botão excluir, a aplicação pedirá confirmação ao usuário se deseja realizar esta operação.
Uma vez efetuado o processo de exclusão, não será mais possível recuperar os dados.
* Recomenda-se executar um backup do usuário antes de executar este processo.
