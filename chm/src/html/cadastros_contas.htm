Contas

Onde encontrar:
Menu Principal -> Cadastros -> Contas.

Descrição:
Abre a janela para a manutenção de contas utilizadas na aplicação.

Imagem da tela:

Itens da tela :
Contas: Lista que contém todas as contas cadastradas.
Ativo: Exibe se a conta selecionada está ativa ou não.
Conta: Descrição da conta selecionada.
Saldo atual: Exibe o saldo atual da conta selecionada.
Limite negativo: Exibe o limite negativo da conta selecionada.
Observações: Exibe as observações da conta selecionada.

Comandos da tela :
Inserir: Limpa a tela para inserção de uma nova conta.
Editar: Permite editar informações da conta selecionada.
Salvar: Grava as informações.
Cancelar: Cancela todas as alterações realizadas.
Excluir: Exclui uma conta selecionada.
Atualizar: Recarrega os dados da tela.
Fechar: Fecha a janela de cadastro de contas.

Menu de contexto:
Contas: Ao clicar com o botão direito do mouse na lista de contas, serão exibidas as seguintes opções:
- Copiar conteúdo: Copia o conteúdo da lista para a área de transferência.
- Exportar para arquivo: Permite exportar o conteúdo da lista para um arquivo do tipo texto (*.txt) ou csv (*.csv).

Observações:
Inserir/Editar: Ao selecionar esta opção, os botões salvar/cancelar serão habilitados e os demais desabilitados.
Excluir: Se houver movimentação na conta selecionada, a aplicação não permitirá que a mesma seja excluída.
Saldo atual: É atualizado automaticamente conforme sejam realizadas as movimentações.
Limite negativo: Permite ajustar um limite máximo de valor negativo para a conta. Caso o saldo atual atingir este limite, não será possível realizar novas movimentações de saída.
Ativo: Permite ativar/desativar uma conta. Ao desativar uma conta, a mesma ficará indisponível para movimentações futuras.
Observações: Quaisquer dados que o usuário deseje colocar a mais, apenas como referência de consulta.
