Lançar Contas a Pagar

Onde encontrar:
Menu Principal -> Financeiro -> Contas a Pagar -> Lançar

Descrição:
Permite realizar o lançamento de uma nova conta a pagar.

Imagem da tela:

Itens da tela :
Vencimento: Data de vencimento do lançamento.
- Lançar múltiplos vencimentos: Define se o lançamento é múltiplo, ou seja, se contém mais de uma parcela.
   - Quantidade: Define a quantidade de lançamentos a ser realizado.
   - Lançar a cada: Define o tempo de lançamento. Valor numérico.
   - Selecione/Meses/Anos: Define o cálculo do tempo para o lançamento.
Tipo de despesa: Permite selecionar qual despesa será utilizada no lançamento.
Valor: Valor do lançamento.
Descrição: Descrição do lançamento.
Forma de pagamento: Permite selecionar qual forma de pagamento será utilizada no lançamento.
Documento: Documento do lançamento.
Código de barras: Código de barras do lançamento.
Observações: Observações do lançamento, ou seja, quaisquer dados que o usuário deseje colocar a mais, apenas como referência.
- Baixar conta imediatamente: Define se o lançamento será baixado imediatamente.
   - Conta: Permite selecionar qual conta será utilizada na baixa imediata.

Comandos da tela :
Lançar: Faz a validação dos dados inseridos nos campos e, em seguida, realiza a lançamento da conta a pagar.
Cancelar: Cancela todas as alterações realizadas fechando a janela posteriormente.

Observações:
- Antes de realizar o lançamento, a aplicação verificará se a data do vencimento é inferior à data atual. Caso positivo, será emitido um aviso informando que não é possível realizar o lançamento com data de vencimento retroativa. Este comportamento pode ser alterado na tela de configurações, selecionando a opção [Permitir lançamentos retroativos].
- Ao marcar a opção [Lançar múltiplos vencimentos] o quadro [Múltiplos vencimentos] será habilitado e a opção [Baixar conta imediatamente] será desmarcada e desabilitada. A aplicação não permite realizar múltiplos lançamentos com baixa imediata, ou seja, para utilizar este recurso, deverá ser feito somente um lançamento por vez.
- Realizando múltiplos lançamentos:
   - Exemplo, lançar uma conta com 6 parcelas, uma por mês:
    - Marcar a opção [Lançar múltiplos vencimentos];
    - Inserir no campo [quantidade] o valor 6;
    - Inserir no campo [lançar a cada] o valor 1 e selecionar a opção [ - meses];
    - A opção [Antecipar pagamento caso data calculada não seja dia útil] altera a data de vencimento para a sexta-feira anterior à data calculada caso a mesma resulte em sábado ou domingo.
- Ao executar o comando de baixa imediata, a aplicação fará a consistência do valor a ser baixado com o limite negativo da conta selecionada para a baixa. Caso o cálculo da baixa não ultrapasse o limite negativo definido para a conta, a operação será concluída. Do contrário, será emitido um aviso informando por qual razão não foi possível realizar a baixa imediata da conta a pagar.
- Ao executar o comando de baixa imediata, caso a opção Considerar data de vencimento na baixa imediata estiver marcada na tela de configurações, a aplicação irá gravar a data selecionada nos campos vencimento e pagamento ao invés da data atual do sistema operacional.
- Ao executar o comando de baixa imediata, caso a opção [Não permitir lançamentos duplicados] esteja marcada na tela de configurações, o sistema irá verificar se já existe baixada na movimentação algum registro com o mesmo valor, mesmo documento e data de pagamento ou vencimento iguais. Se positivo, o sistema irá impedir que a baixa seja feita. Do contrário, a baixa será feita normalmente.
