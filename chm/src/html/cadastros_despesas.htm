Despesas

Onde encontrar:
Menu Principal -> Cadastros -> Despesas.

Descrição:
Abre a janela para a manutenção de despesas utilizadas na aplicação.

Imagem da tela:

Itens da tela :
Despesas: Lista que contém todas as despesas cadastradas.
Fixa?: Sinaliza se a despesa é fixa e está presente pelo menos uma vez ao mês.
Ativo: Exibe se a despesa selecionada está ativa ou não.
Despesa: Descrição da despesa selecionada.
Observações: Exibe as observações da despesa selecionada.

Comandos da tela :
Inserir: Limpa a tela para inserção de uma nova despesa.
Editar: Permite editar informações da despesa selecionada.
Salvar: Grava as informações.
Cancelar: Cancela todas as alterações realizadas.
Excluir: Exclui uma despesa selecionada.
Atualizar: Recarrega os dados da tela.
Fechar: Fecha a janela de cadastro de despesas.

Menu de contexto:
Despesas: Ao clicar com o botão direito do mouse na lista de despesas, serão exibidas as seguintes opções:
- Copiar conteúdo: Copia o conteúdo da lista para a área de transferência.
- Exportar para arquivo: Permite exportar o conteúdo da lista para um arquivo do tipo texto (*.txt) ou csv (*.csv).

Observações:
Inserir/Editar: Ao selecionar esta opção, os botões salvar/cancelar serão habilitados e os demais desabilitados.
Excluir: Se houver movimentação com a despesa selecionada, a aplicação não permitirá que a mesma seja excluída.
Ativo: Permite ativar/desativar uma despesa. Ao desativar uma despesa, a mesma ficará indisponível para movimentações futuras.
Observações: Quaisquer dados que o usuário deseje colocar a mais, apenas como referência de consulta.
