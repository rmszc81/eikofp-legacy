Configurações

Onde encontrar:
Menu Principal -> Configurações.

Descrição:
Permite ao usuário ajustar as configurações da aplicação.

Imagem da tela:

Itens da tela :
- Banco de dados:
    - Permite ajustar configurações de backup da base de dados da aplicação.
    - Período: Define se backup será realizado diariamente, semanal, quinzenal ou mensal.
    - Ativar backup do banco: Selecione esta opção para que os backups sejam realizados automaticamente.
    - Pasta para backup [...]: Local onde a aplicação irá gravar o arquivo de backup, definido pelo usuário.
- Moeda:
    - Permite ao usuário selecionar moedas como Real, Euro, Dólar e Yene.
- Data:
    - Define o período de intervalo de data utilizado na aplicação.
- Configurações gerais:
    - Carregar agenda financeira no Login: permite ao usuário ativar ou desativar o carregamento da agenda financeira ao fazer o login na aplicação.
    - Permitir lançamentos retroativos: permite ao usuário fazer um lançamento em uma data anterior à atual.
    - Não permitir lançamentos duplicados: faz com que a aplicação impeça o usuário de baixar na movimentação lançamentos duplicados *.
    - Permitir alterações de detalhes (movimentação): possibilita que o usuário altere lançamentos já realizados.
    - Considerar data de vencimento na baixa imediata: faz com que a aplicação grave a data selecionada ao invés da data do sistema operacional para os campos vencimento e pagamento (contas a pagar/contas a receber).

Comandos da tela :
Aplicar: Faz a validação das opções selecionadas e, em seguida, grava as configurações.
Cancelar: Fecha a tela de configurações, descartando todas as alterações realizadas.

Observações:
- As configurações são aplicadas imediatamente, não sendo necessário reiniciar a aplicação.
* Lançamentos duplicados são aqueles que contém o mesmo valor, documento e data de pagamento ou vencimento iguais.
