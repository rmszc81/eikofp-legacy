Baixar Contas a Pagar

Onde encontrar:
Menu Principal -> Financeiro -> Contas a Pagar -> Baixar

Descrição:
Permite realizar a baixa de uma conta a pagar lançada anteriormente.

Imagem da tela:

Itens da tela :
Lançar na conta: Permite selecionar a conta na qual será dada a baixa.
Vencimento: Data de vencimento do lançamento.
Descrição: Descrição do lançamento.
Tipo de despesa: Permite selecionar qual despesa será utilizada na baixa.
Valor: Valor do lançamento.
Forma de pagamento: Permite selecionar qual forma de pagamento será utilizada na baixa.
Documento: Documento do lançamento.
Código de barras: Código de barras do lançamento.
Observações: Observações do lançamento, ou seja, quaisquer dados que o usuário deseje colocar a mais, apenas como referência.

Comandos da tela :
Baixar: Faz a validação dos dados inseridos nos campos e, em seguida, realiza a baixa da conta a pagar.
Cancelar: Cancela todas as alterações realizadas fechando a janela posteriormente.

Observações:
- Antes de realizar a baixa, a aplicação verificará se a data do vencimento é inferior à data atual. Caso positivo, será emitido um aviso informando que não é possível realizar a baixa com data de vencimento retroativa. Este comportamento pode ser alterado na tela de configurações, selecionando a opção [Permitir lançamentos retroativos].
- Caso a opção [Não permitir lançamentos duplicados] esteja marcada na tela de configurações, o sistema irá verificar se já existe baixada na movimentação algum registro com o mesmo valor, mesmo documento e data de pagamento ou vencimento iguais. Se positivo, o sistema irá impedir que a baixa seja feita. Do contrário, a baixa será feita normalmente.
- Ao executar o comando de baixa, a aplicação fará a consistência do valor a ser baixado com o limite negativo da conta selecionada para a baixa. Caso o cálculo da baixa não ultrapasse o limite negativo definido para a conta, a operação será concluída. Do contrário, será emitido um aviso informando por qual razão não foi possível realizar a baixa.
