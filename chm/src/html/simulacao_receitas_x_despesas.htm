Receitas x Despesas

Onde encontrar:
Menu Principal -> Simulação -> Receitas x Despesas.

Descrição:
Exibe uma lista de valores simulados, Receitas x Despesas, de acordo com os filtros selecionados.

Imagem da tela:

Itens da tela :
- Filtros:
    - Base de cálculo: Define o período que será utilizado para a simulação.
    - Parâmetros: Contém as listas de Receitas/Despesas que serão utilizadas na simulação.
    - Ordenação dos resultados: Define o campo e a ordem que será utilizada para ordenação dos resultados.
- Rodapé:
    - Exibe valores simulados de receitas/despesas/total de acordo com filtro selecionado.

Comandos da tela :
Iniciar: Limpa todos os campos da tela retornando a mesma ao seu estado inicial.
Filtrar: Exibe a lista de simulação de acordo com os filtros selecionados.
Fechar: Fecha a janela de Simulação de Receitas x Despesas.

Menu de contexto:
Lista de receitas/despesas: Ao clicar com o botão direito do mouse em qualquer uma das listas, serão exibidas as seguintes opções:
- Copiar conteúdo: Copia o conteúdo da lista para a área de transferência.
- Exportar para arquivo: Permite exportar o conteúdo da lista para um arquivo do tipo texto (*.txt) ou csv (*.csv).

Observações:
- A coluna [Simulado] é calculada de acordo com a soma das despesas/receitas selecionadas e posteriormente este valor é dividido pelo período solicitado.
- A coluna [Rec/Pago Mês/Ano] é a total das despesas/receitas selecionadas que foram pagas/recebidas no mês/ano atual.
