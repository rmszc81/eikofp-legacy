Restaurar Backup

Onde encontrar:
Menu Principal -> Backup -> Restaurar backup anterior...

Descrição:
Faz a restauração de um backup da base de dados.

Como proceder para restaurar um backup existente:

1) Ao selecionar o menu, a tela abaixo será exibida. Para iniciar o processo, o usuário deve clicar no botão [...].

2) Em seguida, selecione o arquivo de backup que deseja restaurar.

3) Após a seleção do arquivo, digite a senha do usuário que realizou o backup. Em seguida, clique Restaurar (F2).

4) A aplicação emitirá um aviso sobre o conteúdo da base atual. Clique em Sim para continuar ou Não para cancelar.

5) Uma vez iniciada a restauração do backup, basta apenas que o usuário aguarde a conclusão do processo.

6) Depois do término da restauração, basta clicar em Ok. Pronto! Os dados foram restaurados.

Itens da tela :
Backup a restaurar: Arquivo selecionado para restauração.
Usuário: Usuário que criou o arquivo de backup.
Data do backup selecionado: Exibe a data e hora de quando foi realizado o backup.
Senha: Campo para inserção da senha do usuário que criou o backup.
Log de operações: Exibe uma lista detalhada das operações realizadas no momento.

Comandos da tela :
[...]: Abre a tela para seleção do arquivo a restaurar.
Restaurar: Faz as validações necessárias e inicia o processo de restauração.
Cancelar: Cancela o processo sem restaurar a base de dados.

Barras de progresso :
Acima: Exibe o progresso geral da operação.
Abaixo: Exibe o progresso do item individual.

Observações:
Somente o usuário que criou o backup da base selecionada pode restaurá-la, ou seja, não é possível restaurar uma base de dados de um usuário para outro.
O tempo deste processo pode variar de acordo com o tamanho da base de dados.
Esta tela só poderá ser fechada após a conclusão do processo.
