Criar Usuário

Onde encontrar:
Tela de login -> Criar Usuário

Descrição:
Abre a janela de criação de novos usuários.

Imagem da tela:

Itens da tela :
Nome de usuário: Campo obrigatório, mínimo 02 caracteres.
Senha: Campo obrigatório, mínimo de 04 caracteres.
Confirmar senha: Campo obrigatório, mínimo de 04 caracteres.
Lembrete de senha: Opcional, máximo de 40 caracteres.

Comandos da tela :
Criar: Executa a validação dos campos preenchidos e em seguida, cria o usuário.
Cancelar: Cancela a operação, retornando para a tela anterior.

Observações:
A aplicação não permite que seja criado mais de um usuário com o mesmo nome.
É necessário que os campos [senha] e [confirmar senha] sejam iguais.
O campo [lembrete de senha] é apenas uma dica para o usuário se lembrar da senha, caso venha a esquecê-la.
